# Pencil
A lightweight drawing app optimized for the Apple Pencil.

![alt text](https://codesandbox.io/api/v1/sandboxes/1stnc/screenshot.png "Pencil")




### With support for 3D Touch / Force Touch API Demo
- force
- radiusX 
- radiusY
- touchType
- altitudeAngle
- azimuthAngle
